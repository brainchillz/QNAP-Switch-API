# QSW-M3216R-8S8T — Private REST API v2 reference

Base URL: `http://<switch>/api/v2`  (HTTPS on 443 also works)

Version is reported by unauthenticated `GET /api/about` (`{"ApiVersion":"v2"}`). Auth is identical to v1: `POST /users/login` -> JWT, sent as `Authorization: Bearer <jwt>`.

212 endpoints, reverse-engineered from the QSS web UI bundle (Vite/Vue3 build). This is the v2 superset of [`API.md`](./API.md); notable v2-only areas: `/ip` routing, `/dot1x`, `/arp` + ARP inspection, `/airgap`, `/ipsg`, `/radius`, `/pfc`, `/ptp`, `/certificate`, `/acl/ipv6*`.
